# Print list of Characters

## Quick
### Python
```python
print(list("my string"))
```

### Java
```java
String name = "my string";
char[] charArray = name.toCharArray();
List<Character> listChars = new ArrayList<Character>();
  for (char c : charArray){
    listChars.add(c);
  }
System.out.println(listChars);
```
